# Multi-Threaded Card Game Simulation

A Java-based multi-threaded card game simulation developed as part of a Software Development coursework project.

The system models players, decks, packs, and hands using object-oriented design principles. Each player operates concurrently, drawing and discarding cards whilst attempting to satisfy a winning condition. The project emphasises software architecture, concurrency, testing, and modular design.

## Features

* Multi-threaded player simulation
* Object-oriented architecture
* Custom game logic and win conditions
* File-based game logging
* JUnit unit testing
* Exception handling and validation
* Modular class design using interfaces and abstraction
